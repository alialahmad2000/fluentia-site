# Fluentia Academy — موقع أكاديمية طلاقة

## طريقة النشر السريعة (أقل من 10 دقائق)

### الخطوة 1: إنشاء حساب GitHub
1. ادخل على https://github.com وسجّل حساب جديد (مجاني)
2. اضغط على زر **"+"** أعلى اليمين → **"New repository"**
3. سمّ المشروع: `fluentia-site`
4. اضغط **"Create repository"**

### الخطوة 2: رفع الملفات
1. في صفحة المشروع الجديد اضغط **"uploading an existing file"**
2. اسحب كل ملفات المشروع هنا (package.json, vite.config.js, index.html, ومجلد src)
3. اضغط **"Commit changes"**

### الخطوة 3: النشر على Vercel (مجاني)
1. ادخل على https://vercel.com وسجّل بحساب GitHub
2. اضغط **"Add New Project"**
3. اختر مشروع `fluentia-site` من القائمة
4. Vercel يكتشف تلقائياً إنه مشروع Vite
5. اضغط **"Deploy"**
6. خلال دقيقة يعطيك رابط مثل: `fluentia-site.vercel.app`

### الخطوة 4: ربط دومين خاص (اختياري)
1. في Vercel اذهب لـ Settings → Domains
2. أضف الدومين مثل: `fluentia.sa` أو أي دومين تملكه
3. غيّر DNS عند مزود الدومين حسب التعليمات
